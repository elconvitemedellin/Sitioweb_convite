# El Convite — Sitio web

Landing oficial de El Convite Medellín.

## Stack
- Astro 4 (sitio estático)
- Vercel (hosting)
- Cloudflare (DNS + email)

## Desarrollo local
```bash
npm install
npm run dev
```

## Estructura
- `src/pages/index.astro` — landing principal
- `public/logo.png` — logo horizontal
- `public/favicon.png` — favicon

## Deploy
Push a `main` → deploy automático en Vercel.
